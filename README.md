## 2. Python Mentor

**Описание:**  
AI/EdTech MVP-платформа для обучения Python. Пользователь проходит урок, загружает `.py` файл с решением, после чего система отправляет код в LLM и возвращает разбор.

**Что реализовано / проработано:**

- регистрация и onboarding пользователя
- структура уроков и заданий
- загрузка `.py` файла
- отправка решения в LLM через OpenRouter API
- AI-review: ошибки, рекомендации, статус решения
- сохранение решений
- fallback-логика при недоступности AI-сервиса

**Технологии:**  
Python, FastAPI, Next.js, SQLite, OpenRouter API, HTML, CSS, JavaScript

**Моя роль:**  
Проектировал пользовательский путь, backend-логику, AI-сценарий проверки решений и структуру MVP.

**Репозиторий:**  
https://github.com/Alexandr2807/python-mentor

## 4. AlphaForge Autopilot

**Описание:**  
AI-агент / автономная торговая система для Московской биржи в рамках хакатона. Идея — система анализирует рынок, ищет возможности, оценивает риски и принимает решения в рамках заданных ограничений.

**Что проработано:**

- архитектура AI-агента
- анализ рыночных данных
- поиск торговых возможностей
- риск-контроль
- portfolio/risk слой
- журналирование решений
- fallback-режим без LLM
- ограничения, чтобы AI не мог обходить риск-правила

**Технологии и подходы:**  
Python, API, LLM, risk logic, agent workflow, SQLite/JSONL logs

**Моя роль:**  
Участвовал в проектировании логики агента, риск-модуля, AI-слоя и структуры командной разработки.
